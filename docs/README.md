# Documentation Index

This directory contains the active platform documentation for the Land Feasibility Platform.

## Structure

- `architecture/`
  - platform architecture, runtime shape, service interfaces, and system descriptions
- `agents/`
  - agent role definitions, workflow rules, and orchestration guidance
- `roadmaps/`
  - master roadmap, domain roadmaps, and milestone execution roadmap documents
- `contracts/`
  - canonical data contract references and compatibility documentation
- `system_state/`
  - documentation snapshots and historical state summaries

Additional system-boundary references:

- `repository_structure.md`
- `system_boundaries.md`

## Repository Boundary
